## 6 Lessons

1. **Focus > Features** (Google mastered search)  
2. **Timing > Everything** (missed social/mobile)  
3. **Vision > KPIs** (quarterly chaos killed them)  
4. **Mobile-first required**  
5. **Leadership alignment** matters  
6. **Can't acquire relevance**
